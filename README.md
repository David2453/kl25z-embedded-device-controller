# kl25z-embedded-device-controller
Bare-metal firmware for a smart electric toothbrush on the NXP FRDM-KL25Z (Cortex-M0+). FSM-based architecture with 3 brushing modes, 4-quadrant 2-min timer, FSR pressure sensing, PWM motor control, SSD1306 OLED display (I2C), LED indicators, and interrupt-driven debounced button input.
